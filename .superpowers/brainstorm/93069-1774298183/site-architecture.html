关键交互设计

🎬 滚动动画
GSAP ScrollTrigger 驱动，元素随滚动淡入/视差移动。首页 Hero 的 AI 光球跟随鼠标微动。
✨ 粒子系统
Canvas 绘制的浮动粒子，暗色段密集（科技感），亮色段稀疏渐隐（自然过渡）。
🌓 暗明过渡
每个页面顶部暗色渐变到底部明亮，通过 CSS gradient + ScrollTrigger 背景色插值实现。
📱 响应式
桌面优先，适配平板和手机。移动端导航折叠为汉堡菜单，粒子数量减少以保证性能。
