A. 地图 + 底部半屏列表

上半屏地图显示标点，下半屏可拖拽的列表卡片。点击标点 → 列表自动滚到对应企业。列表可向上拖拽覆盖地图变成全屏列表。

Tab 行为：商品 Tab 地图标点 = 产地/发货地，企业 Tab 标点 = 企业位置。

B. 地图仅限企业 Tab

地图切换按钮只在企业 Tab 下激活。商品 Tab 下按钮置灰或隐藏。逻辑简单——地图本来就是看企业位置的。

优点：概念清晰，商品不需要地图。缺点：限制了功能可用性。

C. 全屏地图 + 浮动卡片（推荐）

地图模式下全屏显示地图，搜索框和 Tab 浮动在地图上方（半透明背景）。点击标点弹出底部浮动卡片，显示企业信息 + 热门商品缩略图。Tab 仍然可切换：企业 Tab 显示企业标点，商品 Tab 显示产地/发货地标点。

优点：地图空间最大化，交互沉浸感强，信息不割裂。
