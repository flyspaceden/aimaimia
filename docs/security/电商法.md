冻鲜 / 冰鲜产品 退换货声明（电商法合规版）
依据《中华人民共和国电子商务法》《消费者权益保护法》《网络购买商品七日无理由退货暂行办法》，冻鲜、冰鲜、冷冻水产 / 肉类等易腐保鲜类商品属于法定不适用七日无理由退货范畴，本店制定如下退换货规则，敬请知悉：
一、适用范围
本声明适用于本店销售的冰鲜、冷鲜、冷冻、速冻等需低温保鲜 / 冷链运输的生鲜食品。
二、不支持无理由退换
因商品易腐、保质期短、冷链依赖度高，退回后无法保障食品安全与二次销售，不支持七天无理由退货、不支持个人原因退换（含不想吃、买多了、口味不符、地址填错、延迟签收、未及时冷冻等）。
已拆封、已解冻、已加工、已食用、包装破损的商品，一律不予退换。
三、质量问题可售后（唯一可退换情形）
符合以下条件，签收后 24 小时内联系客服并提供清晰凭证（商品全貌、快递面单、变质 / 破损 / 漏发照片 / 视频），本店按规定处理：
商品存在变质、异味、严重化冻、包装破损泄漏、重量严重短缺、发错货、漏发等问题；
因物流 / 包装 / 商家原因导致商品损坏；
经核实确属食品安全与品质问题。
四、不予售后情形
非质量问题的个人主观原因；
签收超时、未及时放入冷冻 / 冷藏导致变质；
无法提供有效凭证或超时申报；
已拆封、已加工、已食用；
因买家原因拒收、改地址、无人签收导致损耗。
五、处理标准
质量问题：核实后补发、换货或全额 / 按比例退款；
非质量问题：不予退换，敬请谅解。
六、温馨提示
签收时请当场验货，如有异常请立即拒收并联系客服；
冻品签收后请立即冷冻保存，冰鲜及时冷藏；
本规则符合国家法律规定，下单即视为同意本声明。



《电子商务法》7 天无理由退货合规提示模板
《电子商务法》本身未直接列举7天无理由退货的商品类别，其核心规则由《消费者权益保护法》第25条及配套的《网络购买商品七日无理由退货暂行办法》明确，分为法定绝对不适用与经确认可不适用两类。
一、法定绝对不适用（无条件排除）

以下4类商品，无论商家是否提示，均不适用7天无理由退货：

1. 消费者定作的商品：如定制家具、刻字饰品、个性化服装等。

2. 鲜活易腐的商品：如生鲜、水果、海鲜、鲜花、绿植等。

3. 在线下载或已拆封的数字化商品：如已下载的软件、音乐、视频、电子书、已拆封的音像制品等。

4. 交付的报纸、期刊。

二、经确认可不适用（有条件排除）

以下商品，商家必须在购买时明确提示并经消费者确认，方可不适用7天无理由退货：

1. 拆封后影响安全/品质的商品：如食品、化妆品、贴身内衣、医疗器械、计生用品等。

2. 激活/试用后价值贬损较大的商品：如已激活的手机、电脑、数码产品、大型家电等。

3. 明示的临期/瑕疵商品：销售时已明确告知临近保质期或存在瑕疵的商品。

三、适用前提与商品完好标准

- 适用前提：仅适用于网络、电视、电话、邮购等远程购物，且为个人生活消费用途。

- 时间计算：7天自签收次日起算。

- 商品完好：退回商品需保持原有品质、功能，配件、标识齐全；仅拆封查验不影响完好，但超出查验的使用导致价值贬损则不视为完好。

四、商家合规提示

- 对法定排除商品，必须明确标注，不得隐瞒。

- 对有条件排除商品，必须以显著方式提示并获得消费者确认，否则不能拒绝退货。

- 不得自行扩大排除范围，“一经售出概不退货”等格式条款无效。
