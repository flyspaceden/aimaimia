# 爱买买官网设计方案

## 概述

爱买买品牌官网，面向潜在用户和投资人，展示平台功能、AI 技术能力、团队愿景和商户入驻通道。

**视觉方向：** 暗明交替叙事流 — 暗色科技感开场 → 渐变过渡 → 明亮自然色调，用页面节奏同时兼顾 AI 前沿感和农业温度。

**技术栈：** Vite + React + TypeScript + GSAP (ScrollTrigger) + Tailwind CSS

**素材来源：** Unsplash / Pexels 高质量免费图库

## 页面结构

共 6 个页面，全局共享导航栏和 Footer。

### 全局导航栏

- 固定顶部，滚动时添加毛玻璃背景（`backdrop-filter: blur`）
- 左侧：Logo（渐变圆形）+ 品牌名「爱买买」
- 中间：6 个页面链接（首页 / 产品功能 / AI 技术 / 关于我们 / 商户入驻 / 联系我们）
- 右侧：「下载 App」CTA 按钮（绿色渐变胶囊）
- 移动端：折叠为汉堡菜单

### 全局 Footer

- 暗色背景，与 Hero 段呼应
- 四列布局：品牌信息 / 产品链接 / 技术链接 / 合作链接
- 底部：版权信息 + 隐私政策 + 服务条款

## 交互设计

### 滚动动画（GSAP ScrollTrigger）

- **淡入上移**：所有内容段落默认 opacity:0 + translateY(40px)，进入视口时动画显现
- **视差**：背景图和前景元素不同速率滚动
- **数字递增**：数据亮点段的数字从 0 滚动到目标值
- **背景色插值**：ScrollTrigger 驱动 body 背景色从暗到明的渐变过渡
- **水平滚动**：部分段落（如功能展示）可做横向滚动体验

### 粒子系统（Canvas）

- 暗色段：12-20 个粒子，缓慢浮动，颜色为 AI 青（#00BFA5）和绿（#4CAF50）
- 亮色段：粒子数量递减至 0，透明度渐隐
- 连线：粒子间距 < 120px 时绘制半透明连线（科技网络感）
- 性能：`requestAnimationFrame`，移动端减半粒子数，尊重 `prefers-reduced-motion`

### Hero 光球

- Canvas 或 CSS 实现，径向渐变 + 外层 blur 光晕
- 鼠标跟随：GSAP `quickTo`，延迟 0.3s，移动范围限制在 ±30px
- 呼吸动画：scale 0.95 → 1.05，3s 周期
- 移动端：固定居中，仅保留呼吸动画

### 页面切换

- React Router 路由切换
- 页面进入：整体 fadeIn + 轻微上移（300ms）
- 导航栏：当前页面链接高亮，底部绿色指示点

## 无障碍（Accessibility）

**目标：** WCAG 2.1 AA

### 色彩对比

- 暗底正文文字使用 #B0C4B0 或更亮（对比度 ≥ 4.5:1）
- 亮底正文文字使用 #1A2E1A（对比度 ≥ 7:1）
- 所有 CTA 按钮文字对比度 ≥ 4.5:1

### 动画

- 全局尊重 `prefers-reduced-motion`：检测到时禁用所有 GSAP 动画、粒子、视差、页面过渡
- Canvas 粒子和光球区域添加 `role="img"` + `aria-label` 描述
- 滚动动画退化为直接显示（无 opacity/transform 过渡）

### 键盘与屏幕阅读器

- 汉堡菜单支持键盘操作（Enter 展开/收起，Escape 关闭）
- 所有交互元素有可见 focus 样式（2px 绿色 outline）
- Skip-to-content 链接（首个可聚焦元素）
- 图片一律有描述性 alt 文字

## 性能

**目标：** Lighthouse Performance ≥ 90

### 图片

- 所有图片提供 WebP 格式，`<picture>` 标签回退 JPEG
- 响应式 `srcset`：桌面 / 平板 / 手机三档
- 首屏以下图片全部 `loading="lazy"`
- 单张图片不超过 200KB（WebP 压缩后）

### JavaScript

- 路由级代码分割：`React.lazy` + `Suspense`，每个页面独立 chunk
- GSAP 仅导入 `gsap` 和 `ScrollTrigger`（tree-shake 其余插件）
- 总 JS 体积（gzipped）< 150KB

### 字体

- 使用系统字体栈，不加载网络字体（PingFang SC / Noto Sans SC 均为系统自带）
- `font-display: swap` 作为后备策略

### 其他

- Vite 自动开启 CSS/JS 压缩
- 静态资源添加 hash 文件名（长缓存）

## 联系表单

### 提交方式

使用 Formspree（或同类服务）处理表单提交，无需后端接口：

- 表单 `action` 指向 Formspree endpoint
- 前端处理 loading / success / error 三态 UI

### 表单字段

| 字段 | 类型     | 必填 | 验证规则                                        |
| ---- | -------- | ---- | ----------------------------------------------- |
| 姓名 | text     | 是   | 2-50 字符                                       |
| 邮箱 | email    | 是   | 有效邮箱格式                                    |
| 主题 | select   | 是   | 预设选项：商务合作 / 商户咨询 / 技术支持 / 其他 |
| 消息 | textarea | 是   | 10-1000 字符                                    |

### 防垃圾

- Honeypot 隐藏字段（对机器人可见，人类不可见）
- 提交频率限制提示

### 地图

- 使用高德地图（AMap）静态图片嵌入，不加载交互式 JS SDK
- 点击跳转到高德地图外链
