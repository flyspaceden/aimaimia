# 爱买买 · 商户操作手册（测试版）

> 仅用于测试阶段。含三件事：**如何入驻 · 如何登录 · 如何上货**。

---

## 一、如何入驻

1. 打开官网入驻页：**https://ai-maimai.com/merchants/apply**
2. 填写申请表：
   - 企业/店铺名称（二至五十个字）
   - 经营品类（下拉选择）
   - 联系人姓名
   - **联系手机号**（审核通过后，此号码就是你登录卖家后台的账号）
   - 联系邮箱（选填）
   - 上传营业执照（图片或电子文档，单个文件不超过五兆）
   - 输入图形验证码（看不清可点图片刷新）
3. 点击"提交申请"。
4. 一至两个工作日内，平台管理员审核。
5. 审核通过后，申请时填写的手机号会收到短信通知，即可前往卖家后台登录。

---

## 二、如何登录

1. 打开卖家后台：**https://test-seller.ai-maimai.com/**
2. 页面顶部有两种登录方式，任选其一：
   - **短信登录**：输入手机号 → 点"获取验证码" → 收到短信后填入六位验证码 → 点"登录"。
   - **密码登录**：输入手机号 + 图形验证码 + 密码 → 点"登录"。首次登录请先用短信方式。
3. 如果你的手机号关联了多家企业，登录后会出现"选择企业"页面，点击要进入的那家即可（此页面五分钟内有效，超时需重新登录）。

> 忘记密码、或收不到短信时，请联系平台管理员处理。

---

## 三、登录后如何上货

1. 登录后默认进入**工作台**。点左侧菜单"**商品管理**"。
2. 点右上角"**新增商品**"。
3. 按顺序填写：

   | 填写项 | 说明 |
   |---|---|
   | 商品标题 | 二到八十个字 |
   | 商品分类 | 从树形菜单中选到最末一级 |
   | 商品主图 | 最多九张，第一张是封面 |
   | 商品详情 | 支持图文混排 |
   | 退换货政策 | 如"七天无理由"等 |
   | 自定义属性（选填） | 如"产地=山东" |

4. 往下滚，**添加规格**（也就是每一种口味、重量、包装各一条）：

   | 规格字段 | 说明 |
   |---|---|
   | 规格名称 | 如"一公斤装 / 红富士 / 礼盒装" |
   | 成本价 | 你的进货成本 |
   | 售价 | **系统自动计算 = 成本 × 一点三**，不能手动改 |
   | 库存 | 初始数量 |
   | 单笔限购 | 填 0 表示不限 |
   | 重量（公斤） | 用来算快递费，必须大于零 |

5. 点最底部"**保存**"。
6. 提交后商品进入"**审核中**"，平台管理员通常一个工作日内完成审核。
7. 审核通过后，回到商品列表，把该商品那一行的"**上架开关**"打开，商品就会出现在买家端。

> 审核被驳回时，商品详情页顶部会显示驳回原因；按原因修改后再次保存，会自动重新提交审核。
