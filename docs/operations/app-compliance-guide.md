# 爱买买 App 上架合规指南

> 本文档梳理从零到上架所需的全部行政资质、办理流程与时间线。

## 总览时间线

```
月份    1         2         3         4         5
       ├─────────┼─────────┼─────────┼─────────┼─────────┤
营业执照 ██████░░░░                                        5-15 工作日
ICP备案           ██████████░░░░                           10-20 工作日
软著     ████████████████████████████░░░░                  30-60 工作日（加急7天）
App备案                     ████████████░░░░               10-20 工作日
ICP证                       ██████████████████████░░░░     40-60 工作日
食品经营许可                 ████████████████░░░░           20-30 工作日
上架审核                                      ████████░░░  7-14 天
```

**建议：营业执照 + 软著 + ICP备案 三线并行启动，可将总周期压缩到 3 个月左右。**

## 第一步：注册公司（营业执照）

### 为什么需要
所有后续资质（ICP 备案、ICP 证、App 备案、软著、应用商店开发者账号）都需要企业主体。

### 公司类型选择

| 类型 | 适合场景 | 注册资本建议 |
|------|---------|-------------|
| 有限责任公司 | **推荐**，最常见 | 50-100 万（认缴制，不需实缴） |
| 个体工商户 | 不推荐，无法办 ICP 证 | — |
| 合伙企业 | 多人合伙时考虑 | — |

### 经营范围必须包含
```
技术开发、技术咨询、技术服务；
软件开发；
互联网信息服务；
电子商务；
食用农产品销售；
食品经营（如适用）；
第二类增值电信业务（信息服务业务）
```

> ⚠️ 经营范围中必须有「第二类增值电信业务」才能办 ICP 经营许可证。如果漏了，后续需要去工商变更，会耽误时间。

### 办理流程

1. **核名**（1 天）
   - 登录 [国家企业信用信息公示系统](https://www.gsxt.gov.cn/) 或当地市场监管局网站
   - 提交 3-5 个备选公司名称
   - 格式：`地区 + 字号 + 行业 + 有限公司`，如「杭州爱买买网络科技有限公司」

2. **准备材料**
   - 法人身份证原件 + 复印件
   - 全体股东身份证
   - 注册地址证明（房产证/租赁合同）— 可用商务挂靠地址
   - 公司章程（模板可在当地市监局官网下载）

3. **提交注册**（线上或线下）
   - **线上**：各省「一网通办」平台（推荐）
   - **线下**：当地市场监管局/行政审批中心
   - 审核周期：3-5 个工作日

4. **领取执照后立刻做**
   - 刻章（公章、财务章、法人章、发票章）— 部分地区免费
   - 银行开对公账户（选择网银方便的银行）
   - 税务登记（现在与营业执照同步办理）
   - 社保/公积金开户（有员工时）

### 费用
- 注册登记：免费
- 刻章：200-500 元
- 银行开户：0-500 元（不同银行收费不同）
- 代办全包：800-2000 元（含挂靠地址）

## 第二步：ICP 备案（网站/域名备案）

### 为什么需要
- 所有用中国大陆服务器的域名必须备案
- App 备案的前置条件
- 你的 `api.爱买买.com` 和 `admin.爱买买.com` 等域名都需要

### 前置条件
- ✅ 已有营业执照
- ✅ 已有域名（在国内注册商购买或已转入国内注册商）
- ✅ 已有国内服务器/云主机（阿里云、腾讯云等）

### 办理流程

1. **购买域名**（如果还没有）
   - 在阿里云/腾讯云购买 `.com` 或 `.cn` 域名
   - 完成域名实名认证（与营业执照主体一致）

2. **通过云服务商提交备案**
   - 登录阿里云备案系统：https://beian.aliyun.com/
   - 或腾讯云备案系统：https://cloud.tencent.com/product/ba

3. **填写备案信息**
   - **主体信息**：公司名称、营业执照号、法人信息
   - **网站信息**：域名、网站名称（「爱买买」）、网站用途
   - **服务器信息**：ECS 实例 ID / 备案服务号
   - **负责人信息**：手机号、邮箱、身份证正反面照片

4. **人脸核验**
   - 下载阿里云 App / 腾讯云 App
   - 法人 + 网站负责人分别做人脸识别

5. **等待审核**
   - 云服务商初审：1-2 个工作日
   - 提交管局审核：各省不同，通常 10-20 个工作日
   - 审核期间可能接到管局电话核实，**务必接听**

6. **审核通过**
   - 获得备案号（如：浙ICP备2026XXXXX号）
   - 在网站底部放置备案号并链接到工信部查询页面

### 你需要备案的域名清单
```
爱买买.com          — 主域名
api.爱买买.com      — 后端 API
admin.爱买买.com    — 管理后台
seller.爱买买.com   — 卖家后台
www.爱买买.com      — 官网/落地页
```

> 💡 同一个主域名下的子域名只需要做一次备案，填写主域名 `爱买买.com` 即可。

### 费用
- 备案本身：免费
- 域名：约 50-100 元/年
- 云服务器：根据配置，约 100-500 元/月

## 第三步：软件著作权登记（软著）

### 为什么需要
- 国内安卓应用商店（华为、小米、OPPO、vivo、应用宝等）上架必须提供
- Apple App Store 不要求，但有了更好
- 也是后续申请 ICP 证的加分材料

### 办理流程

1. **准备源代码文档**
   - 提取源代码前 30 页 + 后 30 页（共 60 页）
   - 每页不少于 50 行
   - 去掉空行和注释
   - 页眉标注软件名称和版本号
   - **你的代码已经足够**，从 `backend/src` 和 `app/` 中各取 30 页即可

2. **撰写软件说明书**（操作手册）
   - 约 15-30 页
   - 内容：软件功能介绍 + 操作截图
   - 包含：登录、浏览商品、下单、支付、卖家管理等主要流程截图
   - 参考你的 `docs/architecture/frontend.md` 来写

3. **填写申请表**
   - 登录中国版权保护中心：https://www.ccopyright.com.cn/
   - 注册账号 → 软件著作权登记申请
   - 填写信息：
     - 软件名称：「爱买买农业电商平台软件 V1.0」
     - 开发完成日期：实际完成日期
     - 首次发表日期：可以填计划上架日期
     - 开发方式：独立开发
     - 权利取得方式：原始取得
     - 权利范围：全部权利
     - 编程语言：TypeScript、JavaScript
     - 运行平台：Android、iOS
     - 软件用途：电子商务

4. **提交材料**
   - 可以网上提交（电子版）
   - 也可以邮寄纸质材料到中国版权保护中心

5. **等待审核**
   - 普通流程：30-60 个工作日（免费）
   - 加急流程：
     - 31-35 个工作日：约 ¥600
     - 21-25 个工作日：约 ¥1,500
     - 10-15 个工作日：约 ¥3,000
     - 3-5 个工作日：约 ¥5,000-8,000
   - 代理机构可办加急

### 费用
- 官方登记费：免费（2017 年起免征）
- 代理费（如果找代理）：500-1500 元
- 加急费：另算

## 第四步：App 备案

### 为什么需要
- 工信部 2023 年 8 月发布《工业和信息化部关于开展移动互联网应用程序备案工作的通知》
- 2024 年 4 月起，未备案 App 不得上架

### 前置条件
- ✅ 已有营业执照
- ✅ 已完成 ICP 备案（域名备案）
- ✅ App 已基本开发完成（需要填写功能描述）

### 办理流程

1. **选择备案入口**
   - 方式一：通过已备案的云服务商（阿里云/腾讯云）提交 — **推荐，流程更顺**
   - 方式二：通过应用商店提交（华为、小米等部分商店已接入）
   - 方式三：直接在工信部备案系统提交

2. **通过阿里云提交（推荐）**
   - 登录阿里云 App 备案系统
   - 路径：控制台 → 备案 → App 备案

3. **填写 App 信息**
   - App 名称：爱买买
   - App 类型：Android + iOS
   - App 包名（Android）：`com.aimaimai.app`（你的实际包名）
   - Bundle ID（iOS）：`com.aimaimai.app`
   - App 公钥（Android）：从签名证书提取
   - App SHA-1 签名（Android）：从签名证书提取
   - 所属 ICP 备案号：第二步获得的备案号
   - 服务内容分类：电子商务

4. **获取签名信息**（Android）
   ```bash
   # 生成签名文件（如果还没有）
   keytool -genkey -v -keystore aimaimai-release.keystore \
     -alias aimaimai -keyalg RSA -keysize 2048 -validity 10000
   
   # 查看 SHA-1
   keytool -list -v -keystore aimaimai-release.keystore -alias aimaimai
   
   # 查看公钥
   keytool -list -rfc -keystore aimaimai-release.keystore -alias aimaimai
   ```

5. **提交审核**
   - 云服务商初审：1-2 个工作日
   - 管局审核：10-20 个工作日

6. **审核通过**
   - 获得 App 备案编号
   - 在 App 的「关于」页面展示备案编号

### 费用
- 免费

## 第五步：ICP 经营许可证（增值电信业务经营许可证）

### 为什么需要
- 爱买买是多商户电商平台（商家入驻模式），属于**经营性互联网信息服务**
- 根据《互联网信息服务管理办法》，经营性互联网信息服务必须取得 ICP 证
- 平台抽佣、收取技术服务费等行为均属经营性质

> ⚠️ **ICP 备案 ≠ ICP 证**。备案是告知性的，所有网站都要做；ICP 证是许可性的，只有经营性网站需要。

### 前置条件
- ✅ 已有营业执照（经营范围含「第二类增值电信业务」）
- ✅ 已完成 ICP 备案
- ✅ 注册资本 ≥ 100 万元（认缴即可，不需要实缴）
- ✅ 公司至少有 3 名员工的社保记录（部分省份要求）

### 需要的材料

1. **公司材料**
   - 营业执照副本彩色扫描件
   - 法人身份证正反面
   - 公司章程（工商调档盖章版）
   - 股权结构说明（如有外资股东，需要额外审批）

2. **技术材料**
   - 3 名技术人员的身份证 + 社保证明 + 学历证明
   - 网络与信息安全保障措施（模板可在通信管理局官网下载）
   - 服务器托管协议或云服务合同

3. **业务材料**
   - 业务发展和实施计划（商业计划书简版）
   - 用户服务协议
   - 已备案的域名证书

### 办理流程

1. **网上提交**
   - 登录工信部电信业务市场综合管理信息系统
   - https://tsm.miit.gov.cn/
   - 选择「第二类增值电信业务」→「信息服务业务（仅限互联网信息服务）」

2. **填写申请**
   - 按系统提示填写公司信息、业务信息
   - 上传所有扫描件

3. **省通信管理局受理**
   - 初审：5 个工作日内决定是否受理
   - 审核：受理后 40-60 个工作日
   - 可能要求补充材料或现场核查

4. **领取许可证**
   - 审核通过后，前往省通信管理局领取纸质证书
   - 或邮寄

### 费用
- 官方：免费
- 代办（推荐，流程较复杂）：¥5,000 - ¥15,000
- 如果要 3 名员工社保：按当地社保标准

### 注意事项
- **外资限制**：如果公司有外资股东，ICP 证需要走商务部审批，流程更长
- **年检**：每年 1-3 月需要做年检（ICP 证年报）
- **不办会怎样**：被查到罚款 1-10 万元，严重的吊销营业执照

## 第六步：食品相关资质（如适用）

### 是否需要
爱买买销售农产品，需要区分：

| 品类 | 是否需要食品经营许可证 |
|------|---------------------|
| 初级农产品（生鲜蔬果、生肉、鲜蛋） | **不需要**，但需要食用农产品经营备案 |
| 预包装食品（零食、干货、罐头） | **需要**食品经营许可证 |
| 散装食品 | **需要**食品经营许可证 |
| 特殊食品（保健品、婴幼儿食品） | **需要**食品经营许可证 + 额外批文 |

### 作为平台方
- 爱买买作为平台方，如果**自身不直接销售食品**（只提供平台服务），平台本身可不办食品经营许可证
- 但**入驻商户必须持有**相应食品经营许可证，平台有审核义务
- 如果平台公司自营（`Company.isPlatform=true`），且自营商品包含食品，则平台公司也需要办

### 办理流程（如需要）

1. **准备材料**
   - 营业执照
   - 经营场所平面图
   - 食品安全管理制度文本
   - 从业人员健康证明

2. **提交申请**
   - 到当地市场监管局或网上提交
   - 选择「仅销售预包装食品」可走简化备案

3. **审核**
   - 仅销售预包装食品：备案制，当场办结或 5 个工作日
   - 其他：需要现场核查，20 个工作日

## 第七步：应用商店上架

### Apple App Store

1. **注册开发者账号**
   - 访问 https://developer.apple.com/
   - 选择「组织/公司」类型
   - 需要 Apple ID + D-U-N-S 编号
   - 费用：¥688/年

2. **获取 D-U-N-S 编号**
   - 免费申请：https://developer.apple.com/enroll/duns-lookup/
   - 邓白氏会电话核实公司信息
   - 周期：3-14 天

3. **构建并上传 App**
   ```bash
   # 使用 EAS Build（Expo）
   npx eas-cli build --platform ios --profile production
   
   # 上传到 App Store Connect
   npx eas-cli submit --platform ios
   ```

4. **在 App Store Connect 填写信息**
   - App 名称、描述、关键词、截图
   - 隐私政策 URL（必须）
   - 年龄分级
   - 选择价格（免费）
   - 上传截图：6.7"、6.5"、5.5" 各尺寸

5. **提交审核**
   - 首次审核：通常 1-3 天
   - 常见拒绝原因：
     - 缺少隐私政策
     - 登录功能没有提供测试账号
     - 支付没有使用 IAP（注意：实体商品支付可以用第三方支付，不需要 IAP）
     - 界面有明显 Bug

6. **审核通过 → 上架**

### 国内安卓应用商店

每个商店需要**单独注册、单独上架、单独审核**，主要商店：

| 商店 | 注册地址 | 开发者费用 | 特殊要求 |
|------|---------|-----------|---------|
| 华为应用市场 | developer.huawei.com | 免费 | 需要软著 |
| 小米应用商店 | dev.mi.com | 免费 | 需要软著 |
| OPPO 软件商店 | open.oppomobile.com | 免费 | 需要软著 |
| vivo 应用商店 | dev.vivo.com.cn | 免费 | 需要软著 |
| 应用宝（腾讯）| open.qq.com | 免费 | 需要软著 |
| 豌豆荚/PP助手 | — | — | 已被收编，通过上述商店分发 |

### 每个安卓商店的通用上架流程

1. **注册企业开发者账号**
   - 营业执照
   - 法人身份证 / 授权书
   - 对公账户验证（部分商店）

2. **准备上架材料**
   - APK/AAB 安装包
   - 软著证书扫描件
   - App 备案编号
   - App 图标（512×512）
   - 截图（至少 3-5 张）
   - 隐私政策 URL
   - 应用描述、更新日志
   - 应用分类：购物

3. **构建 Android 包**
   ```bash
   # 使用 EAS Build（Expo）
   npx eas-cli build --platform android --profile production
   ```

4. **提交审核**
   - 各商店审核周期：1-5 个工作日
   - 首次上架审核较严，更新版本会快一些

### 必须准备的法律文档

你需要在 App 内和官网放置以下文档：

1. **隐私政策**（所有商店必须）
   - 收集了哪些个人信息
   - 如何使用、存储、共享
   - 用户权利（查看、删除、注销账号）
   - 参考《个人信息保护法》要求

2. **用户服务协议**
   - 平台与用户的权利义务
   - 交易规则
   - 争议解决

3. **营业执照展示**
   - App 内「关于我们」页面展示

## 第八步：其他可选但建议办理的资质

| 资质 | 是否需要 | 说明 |
|------|---------|------|
| 网络安全等级保护（等保） | 建议二级 | 用户量达到一定规模后建议做，电商平台一般二级 |
| EDI 许可证 | 可能需要 | 如果平台提供在线数据处理与交易处理服务 |

| 广告发布登记 | 不需要 | 除非平台有广告业务 |

## 执行清单（按优先级排序）

```
Week 1-2:
  □ 注册公司（经营范围包含增值电信业务）
  □ 购买域名 + 开始域名实名认证
  □ 购买阿里云 ECS（如还没有）
  
Week 2-3:
  □ 启动 ICP 备案（通过阿里云）
```

## 常见问题

**Q: 没有 ICP 经营许可证能先上架吗？**
A: 技术上可以先上架（App 备案和 ICP 备案是上架的硬性要求，ICP 证不是上架的直接前置条件）。但作为平台型电商，法律上需要持有 ICP 证才能合规运营。建议先上架，同时加紧办理 ICP 证。

**Q: 中文域名（爱买买.com）备案有问题吗？**
A: 没有问题，中文域名可以正常备案。但建议同时注册英文域名（如 `aimaimai.com`）作为技术域名使用。

**Q: 个人开发者能上架吗？**
A: Apple App Store 个人开发者可以上架。国内安卓商店对个人开发者限制较多，强烈建议用企业身份。

**Q: 找代办靠谱吗？**
A: ICP 证建议找代办（流程复杂）。软著可以自己申请也可以代办。ICP 备案和 App 备案通过云服务商系统自助操作即可。

**Q: 外资公司有什么特殊要求？**
A: 如果公司有外资成分（包括港澳台），ICP 证需要商务部审批，流程更长（3-6 个月），且部分业务可能受限。尽量使用纯内资公司申请。
